# 📚 AI Study Planner

AI Study Planner is a simple Streamlit application that generates a personalized study schedule for upcoming exams using the OpenAI API.

The user selects an exam date, enters subjects, and specifies how many hours they can study each day. The application then uses an AI model to generate a structured study plan including subject priorities, a daily schedule, revision strategy, and a final revision plan.

## 🚀 Features

- Generate AI-based study plans
- Calculates remaining days until the exam
- Subject priority ranking
- Daily study schedule
- Weekly revision strategy
- Final revision plan before the exam

## 🛠 Tech Stack

- Python
- Streamlit
- OpenAI API
- python-dotenv
